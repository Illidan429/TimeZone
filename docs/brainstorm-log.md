## 2026-04-02

### 时间与角色

- **日期**：2026-04-02  
- **需求方**：站长（不写代码，只提需求与拍板）  
- **协作方式**：先头脑风暴定方向，再分阶段实现

### 内容：站点定位

- **给新观众**：进站能感到主播**直播做得认真**，更容易被吸引、愿意关注。  
- **给老观众**：互动更好；其中包含 **匿名信箱「棉花糖」**（名称已定）。  
- **对标参考**：[Laplace](https://laplace.live) 类站在「动态 / 周表与录播 / 歌切 / 工具」等方面较完备，本站**不必功能一一对齐**，以真实维护成本与主播需求为准。

### 内容：已确认的模块（首期共识）

| 模块 | 说明（当日理解） |
|------|------------------|
| 主播动态 | 链接或聚合展示主播 **B 站全部动态** |
| 录播表 | 主播自行上传直播录像至 B 站；本站 **抓取封面、标题** 等展示，**视频**视情况 **跳转 B 站** 或 **站内嵌入播放**（待决） |
| 听歌 | 歌切等内容，**主要由站长维护**；站长亦有 B 站账号用于歌切与二创 |
| 工具 | **面向所有人**（含其他主播），相对独立板块，以下载/说明为主；例：Virtual Keyboard（见 [related-repositories.md](related-repositories.md)） |
| 公告 | 站长以**管理员身份**发布 **开发日记 / 站务说明** 等 |
| 棉花糖 | 匿名信箱，服务老观众互动；**运营规则（审核、留存、防滥用）** 待细化 |
| 二创合集 | 以 **B 站视频** 形式策展展示 |

### 内容：信息呈现与换皮

- **首页**：**欢迎图 + 气泡导航**（早期示例含：录播 / 听歌 / 工具）；模块多于 3 个时，可用 **分组气泡** 或 **二级页** 收纳（如「直播相关」下挂周表+录播）。  
- **节日 / 特殊日**：希望 **画风或主图便于更换**，实现上倾向 **资源与配置可替换**，少改代码。

### 内容：边界与公开性

- **暂无**「不想公开」的板块；若上线棉花糖，仍建议在公告中提供 **使用说明**（与是否匿名、审核相关）。

### 内容：基础设施（当日记录）

- **域名**：`timezone.fan` **备案未完成**，上线顺序待与域名就绪情况对齐。  
- **服务器**：阿里云 ECS，**Ubuntu 22.04**，约 **2 核 4G**，系统盘约 **40GiB**，公网带宽 **按使用流量**。**接入信息（含公网 IP）仅存控制台或私有运维笔记，不写入公开仓库**，避免被扫。

### 讨论中提出的扩展（未强制纳入首期）

- **直播周表 / 预告**：与录播表互补；若做，建议与录播同属「直播相关」入口，避免顶级导航过多。  
- **醒目入口**：如「关注 / 进入 B 站空间」主按钮，降低新观众迷路。  
- **工具页**：对外站主播友好时，加强 **说明 + 下载 + 更新日志**。

### 待决问题（留给后续会话）

- 棉花糖：**最小可行规则**（是否仅文字、审核周期、保存时长等）。  
- 录播：**首期是否仅外链 B 站播放**，站内嵌入是否二期再做。  
- **周表**是否进入第一期范围。
