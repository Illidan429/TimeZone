## 占位路由与模块（当前草案）

实现时可改名或合并；与 [brainstorm-log.md § 2026-04-02](brainstorm-log.md#2026-04-02) 对齐。

| 模块 | 占位路径（示例） | 备注 |
|------|------------------|------|
| 首页 | `/` | 欢迎图 + 气泡（可多级或分组） |
| 主播动态 | `/dynamics` 或 `/bilibili/feed` | 指向/聚合 B 站动态 |
| 录播表 | `/archive` 或 `/vod` | 元数据来自 B 站投稿 |
| 直播周表（若做） | `/schedule` | 可与录播同父级入口 |
| 听歌 | `/music` 或 `/songs` | 站长维护 |
| 二创合集 | `/fanworks` 或 `/derivative` | B 站视频策展 |
| 工具 | `/tools` | 对外说明 + 下载/外链 |
| 公告 | `/news` 或 `/blog` | 站长图文 / 开发日记 |
| 棉花糖 | `/marshmallow` 或 `/fm` | 匿名信箱，规则待定 |
| 管理后台（若需要） | `/admin` | 公告、歌单、棉花糖审核等 |
