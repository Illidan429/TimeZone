# 信息架构草案

> 路由与命名仅为占位，实现时可按框架调整。

```
/                     首页（欢迎图 + 气泡导航）
/archive 或 /vod      录播日程 / 回放列表（自动聚合）
/music 或 /songs      歌切 / 音频（管理员维护）
/tools                小工具入口（链接或嵌入键盘等）
/news 或 /blog        可选：站务日志 / 公告 / 长文
/admin                可选：后台（上传音频、编辑公告等，需鉴权）
```

## 气泡与页面映射（示例）

| 气泡文案（示例） | 目标 |
|------------------|------|
| 录播 | `/archive` |
| 听歌 | `/music` |
| 工具 | `/tools` |
| 日志 | `/news` |
